# Todo-Demo

This Simple Todo Project demonstrate the use of Ionic, Cordova, Angular, fetch & express server with swagger API

## express-api

How to run?

```bash
cd express-api
npm install
npm run serve
```

## ionic-client

How to run?

```bash
cd ionic-client
ionic serve -l
```
